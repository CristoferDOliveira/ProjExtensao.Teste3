# LOGIN-TEST-SITE

## Description
This is a project developed to train my programming skills, using JavaScript and Python to create a simple login system. The site does not feature password encryption because I only learned about security standards after the project's creation.

## Tools/Technologies
- VS Code
- GitHub
- Flask
- React (Vite)
- PostgreSQL

## Languages
- Python (Backend)
- JavaScript (Frontend)
- HTML/CSS
- SQL (Database)

## How to Run
1. Clone the repository.
2. Install Flask: `pip install flask` (and any other requirements).
3. Set up your PostgreSQL database.
4. Run the Python server: `python app.py`.
5. Run the React frontend: `npm install` then `npm run dev`.
